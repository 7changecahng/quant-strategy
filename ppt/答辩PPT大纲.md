## 第 3 页 · 选题背景 (1 分钟)

### 为什么做这个项目?

- **A 股投资核心难题**: 何时买? 何时卖?
- **单一指标局限**: 纯均线震荡市假信号多 → 频繁交易 → 手续费侵蚀
- **课程 L11 启发**: "组合指标过滤假信号 + 动态仓位"
- **多维度分析需求**: 趋势 + 动能 + 超买超卖 + 波动 + 量价

### 核心竞争力

```
五维度 AND 复合信号 → 降低假信号 → 低换手高胜率
+ 止损止盈风控 → 回撤可控
+ 参数自动优化 → 避免过拟合
```

### 相关工作
- Murphy (1999): 技术指标组合的有效性
- 熊熊等 (2012): 多指标组合优于单一 MACD

## 第 5 页 · 方法详解 (2 分钟)

### 五层 AND 复合信号

```
signal = (MA5 > MA20)           ← 趋势:短期向上
     AND (MACD_hist > 0)        ← 动能:动量为正
     AND (RSI < 70)             ← 过滤:未超买不追高
     AND (close > BB_lower)     ← 确认:未极端弱势
     AND (volume_ratio > 0.7)   ← 配合:量价不背离
```

### 动态仓位管理

```
position = signal × max(0.5, 1.0 - (RSI-60)/40)
```

RSI > 60 逐步减仓至 50% 底仓 → 过热自动降风险

### 止损止盈机制 🆕

| 类型 | 规则 | 作用 |
|------|------|------|
| 固定止损 | 单笔亏损 ≥ 8% | 限制最大亏损 |
| 固定止盈 | 单笔盈利 ≥ 25% | 锁定利润 |
| 追踪止损 | 从最高点回撤 ≥ 6% | 保护浮盈 |

### 回测引擎 ⚠ 核心

```python
position = np.roll(signal, 1)  # ← 今天信号,明天交易
position[0] = 0                # ← 第一天空仓
# 每次调仓扣 0.05% 手续费
```

**防未来函数三原则**:
1. `shift(1)` — 所有信号滞后一天生效
2. 手续费实时扣除
3. 买入持有基准 — 无对照无结论

## 第 9 页 · 结论与局限 (1 分钟)

### 核心结论

1. ✅ **复合信号优于单一指标**: 收益/风险/效率三维度全面胜出
2. ✅ **动态仓位有效**: RSI 减仓 + 止损止盈将回撤控制到极低水平
3. ✅ **低换手是核心优势**: 年均 3-4 次交易,手续费影响最小化
4. ✅ **初步稳健性**: 10 标的 + 样本外全正超额
5. ✅ **工程化完善**: 39 测试 + 双数据源 + HTML报告 + Plotly Dashboard

### 局限性 (诚实陈述)

| 局限 | 说明 |
|------|------|
| 单边做多 | 熊市自然空仓,不能通过做空获利 |
| 样本局限 | 4年数据未经历 2008/2015 级极端熊市 |
| 过拟合风险 | 样本外 Sharpe 从 1.85 → ~1.3 衰减 |
| 滑点未建模 | 固定费率≈，大单真实成本更高 |

### 未来改进
- 扩展到全部沪深 300 → 截面统计验证
- 多资产风险平价配置 (股+债+商品 ETF)
- LSTM/Transformer 替代 LogisticRegression
